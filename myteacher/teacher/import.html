CSV Import Instructions & Rules

File Requirements

  • File must be .csv format (comma-separated values)
  • Maximum file size: 5MB
  • Maximum 300 data rows (excluding header)
  • First row must be the column headers
  • Use double quotes around fields that contain commas

Question Type Rules

MCQ (Multiple Choice — single answer)
  • correct must be a single letter: A, B, C, D, E, or F
  • The correct letter must have a corresponding option with text
  • Minimum 2 options (A and B), maximum 6
TF (True / False)
  • Set question_type to TF
  • option_a = True, option_b = False
  • correct must be A (True) or B (False)
  • Options C–F are ignored
SATA (Select All That Apply)
  • Set question_type to SATA
  • correct must be comma-separated letters, e.g. A,C,E
  • At least 2 correct answers required
  • Each correct letter must have a corresponding option with text

Import Behaviour

  • Only valid rows are imported — errors and duplicates are skipped
  • Imported items go to your Question Bank with source type "IMPORT"
  • You can then add them to any quiz draft from the bank
  • Column headers are case-insensitive and spaces are treated as underscores
  • If question_type is omitted, it defaults to MCQ
  • If marks is omitted, it defaults to 1
